Installation PIN

You either did not enter the installation PIN or entered an incorrect installation PIN. The installation PIN appears in the server's standard output (console, ssh shell, log file, etc.) as "Install PIN: 000000000"
If you're using pythonanywhere, the Install PIN appears in the server log.
If more than one PIN appears in the log file, use the last one.
